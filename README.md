# Hospitality Analytics Dashboard

## 📌 Project Overview
This project focuses on analyzing hospitality industry data to uncover trends in bookings, revenue, occupancy, and customer behavior. The goal is to support data-driven decision-making for hotels and hospitality businesses.

## 🏨 Business Problem
Hospitality businesses face challenges such as fluctuating occupancy rates, seasonal demand, and revenue optimization. This project analyzes historical data to identify patterns and provide actionable insights.

## 🛠 Tools & Technologies
- Power BI
- SQL
- Python
- Excel

## 📊 Key Analysis Performed
- Booking trends analysis (daily, monthly, seasonal)
- Occupancy rate and revenue performance
- Customer segmentation based on booking behavior
- Cancellation rate analysis
- Average Daily Rate (ADR) and RevPAR insights

## 📈 Key Insights
- Identified peak and off-peak booking seasons
- Highlighted high-revenue room types and customer segments
- Detected high cancellation periods impacting revenue
- Provided insights to improve pricing and capacity planning

## 🚀 Outcome
The analysis helps hospitality managers optimize pricing strategies, improve occupancy rates, and enhance overall revenue performance.
